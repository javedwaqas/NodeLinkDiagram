Aggregate Node-Link Diagram

Waqas Javed, Niklas Elmqvist, Jean-Daniel Fekete

  • Launch the Aggregate Node-Link Diagram demo!
  • Accepts a GraphML file as input. Here is an example file.
  • Uses the Noack LinLog layout (more to be added).
  • Left-click on nodes to expand them.
  • Use the mouse wheel to expand and aggregate the hierarchy.
  • Left-drag to pan, right-drag to zoom.
  • Use the numeric keys (0-9) to change the aggregation level: 1 = 10% of all nodes visible, 9 = 90%, 0 = 100%.
